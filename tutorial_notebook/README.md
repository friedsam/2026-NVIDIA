## 3. GPU & Hybrid Implementation
### Milestone 2: GPU Acceleration
We refactored the $O(N^2)$ energy function to utilize **CuPy** kernels on the **NVIDIA L4**. 
* **Vectorization:** Parallelizing the autocorrelation shifts reduced iteration time by >90% compared to CPU loops.
* **Validation:** ![GPU Utilization Proof](./nvidia_smi_screenshot.png)
*Figure 1: nvidia-smi confirming L4 GPU active VRAM and utilization.*

### Milestone 3: Quantum Tiling Strategy
To overcome qubit limitations ($N=20$), we implemented a **Periodic Tiling** strategy. We used a Trotterized circuit to sample the energy landscape, then tiled the result to $N=40$. While this created initial "seam" artifacts (higher starting energy), the structural patterns provided a robust basin for the GPU solver to reach a final energy of 124.0.
